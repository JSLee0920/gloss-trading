# ADR-006 — The project is called Gloss

## Decision

**Gloss.** Previously working-titled *Hindsight*.

## Rationale

A gloss is an explanatory note written beside a difficult passage. The trades are the passage; the app writes in the margin.

The metaphor already runs through the whole design system — chess annotation, proofreader's ink, marks that point at a passage without rewriting it. See the annotation mark as signature element in [[Socratic Replay]].

It also encodes the product's core stance: **a gloss never replaces the text, it comments alongside it.** That is [[Non-Defensive Diff]] stated in one word.

## Why not Hindsight

"Hindsight is 20/20" is a phrase about knowledge arriving too late to be useful — the opposite of the pitch. It is also a crowded namespace.
