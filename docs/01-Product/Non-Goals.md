# Non-Goals

Explicit. Revisit only with a written reason.

- **No price predictions, no buy/sell signals.** The system analyses and explains. This is a positioning choice as much as a compliance one — the moment it predicts, it is judged on accuracy instead of insight.
- **No trade blocking.** It advises, it does not restrict. Nudges have cooldowns. See [[Nudge Policy]].
- **No profit promises.** Content teaches concepts and analysis.
- **No copy trading, no social feed, no leaderboard.**
- **No broker order execution.** Read-only, always.
- **No mobile app** until the web loop retains people.
- **No multi-broker OAuth** — statement import covers everyone. See [[ADR-004 No Broker API]].
- **No sentiment score without provenance.** Sentiment is reported as attributed counts with sources. See [[Market Analysis]].
- **No auto-publishing.** Every generated public post passes anonymisation and human approval. See [[Content Studio]].
- **No celebrating profitable outcomes.** Only held process. See [[Positive Patterns]].
- **No LLM-invented concepts.** See [[ADR-005 Fixed Concept Taxonomy]].
